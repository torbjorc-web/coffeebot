Coffee Chatbot
This is a Python chatbot that takes coffee orders in advance.

Features
Chooses drink size

Chooses drink type

Handles latte milk options

Bonus: asks for cup type

Bonus: asks if the drink should be hot or iced

Bonus: allows ordering an extra drink

How to run
bash
python3 script.py
Files
script.py: main chatbot program

README.md: project description and instructions
